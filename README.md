# Introduce
BlockPJDBC is sub project for BlockP Projects. 

# Features  
## Project
>project type:jdbc  
## Format
>format register sql/jdbclink(.bpjdbc)  

## Function
>connect to database by jdbc driver(auto load ojdbc8.jar/sqlite-jdbc.jar/postgresql.jar in ../libs/)  
>load db structure cache in jdbc project  

## Transformer
>transformer for blob>byte[]  
>transformer for clob>String
